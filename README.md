## _build

```bash
# NPM
$ npm run build:app
# YARN
$ yarn build:app
```

## _build desktop app

```bash
# NPM
$ npm run build:native:{all|linux|windows|macos}
# YARN
$ yarn build:native{all|linux|windows|macos}
```

## _dev

```bash
# NPM
$ npm run serve
# YARN
$ yarn serve
```
