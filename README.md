## Installation
---

Codepad has not been packaged for Windows, Mac or Linux yet. But you can run it via CLI by following these steps:
 - Install npm
 - Install Electron: `npm install electron -g`
 - Clone this repository
 - `cd codepad`
 - `electron .` or `npm start`
